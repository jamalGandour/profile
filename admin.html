Hostinger Checklist

Before Updating From GitHub

  1. Confirm your live database is backed up.
  2. Confirm `DB_PATH` points to a persistent folder outside the GitHub deployment folder.
  3. Recommended path: `/home/YOUR_USER/profile-data/jamal-profile.db`.
  4. Download SQLite backup from this tab before major changes.
  5. After deployment, return here and check that Database Mode says `External persistent folder`.
